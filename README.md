# Green Catalyst Services Multi-Page Website

Updated from one-page website into a multi-page static website.

## Pages
- `index.html` — Home page with Hero + Why Green Catalyst
- `services.html` — Services
- `portfolio.html` — Portfolio gallery
- `process.html` — Process
- `contact.html` — Contact CTA

## Contact CTA
The website no longer displays the email address or phone number openly in contact cards.
It uses action buttons:
- Request via WhatsApp
- Email Us
- Call Us

## SEO
- Each page has its own title and meta description.
- `sitemap.xml` includes all pages.
- `robots.txt` points to the sitemap.
- Home page includes LocalBusiness structured data.

## Deploy
Replace the files in your GitHub repository with the contents of this folder.
Commit and push using GitHub Desktop. Vercel will auto-redeploy.
